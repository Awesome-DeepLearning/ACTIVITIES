## FAQ
- 我可以向自己的仓库提交 PR 吗？  
答：不可以！必须官方仓库  
- 我提交的 PR 必须要合并后才算吗？  
答：只要提交 PR 即可积分，除非被标记为无效的。  
- Issues 算数吗？  
答：不算！  
- 我可以向 GitHub 以外的平台提交 PR 吗？  
答：不可以！
